## PROJECT INTRODUCTION
<a name="gioithieumonhoc"></a>
### Back-End
+ Our team configure the database on MongoDB
+ Our team build our server using ExpressJS
+ Our team deploy it on Heroku.

### Front-End
+ Our team use Figma to design UI
+ Our team develop AInE as an Android app using React Native Framework and JavaScript programming

### Members Management
+ Our team use Scrum model to manage the AInE's development progress
+ Our team conduct task scheduling by Trello
